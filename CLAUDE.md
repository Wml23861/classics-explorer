# classics-explorer — 跨会话连续性

## 会话启动时必须执行

1. 检查 `.claude/.session-ended` 是否存在 — 如果存在，说明上次会话异常结束未保存摘要，请在 SESSION_LOG.md 中询问用户并补写
2. 读取 `.claude/SESSION_LOG.md` 了解之前对话的关键上下文
3. 读取 `.claude/MEMORY.md` 了解项目记忆索引
4. 读完日志后，在回复中简要确认你已了解之前的上下文

## 会话结束前必须执行

在会话自然结束或用户表示要离开时，**主动将本次对话的关键内容追加到 `.claude/SESSION_LOG.md`**，格式：

```
## [日期] 会话摘要
- 主要话题：
- 关键决策/进展：
- 下次需跟进的事项：
```

保存完成后，删除 `.claude/.session-ended` 标记文件。
不要等用户提醒，主动保存。

## 重要信息持久化

对于跨会话都需要记住的信息（用户偏好、项目决策、个人背景等），写入 `.claude/memory/` 对应类型的记忆文件，并更新 `MEMORY.md` 索引。
